# JS Weather App

Js Weather App is a practice project built as part of [The Odin Project's JavaScript Course](https://www.theodinproject.com/lessons/node-path-javascript-weather-app). The application allows users to search for any location and view its current weather conditions and forecast. It displays information such as temperature (with support for both Celsius and Fahrenheit), wind speed, humidity, and other key weather data through a clean and responsive interface.

## Features
- Search weather by location
- Display current weather conditions
- Toggle between Celsius and Fahrenheit
- View Humidity, wind speed, and other weather metrics
- Responsive an user-friendly interface
